каждая команда начинается с git
git log - просмотр последних действий
git status - просмотр файлов на локальном диске (какие есть)
git commit -m "Описание файлов"
git add . - подготовить файлы к отправке
git push - отправить на сервер гитхаба
СОЗДАНИЕ НОВОЙ ВЕТКИ
git checkout -b название ветки - переход в ветку
в новой ветке можно создавать файлы, при переходи в ветку файлы на локалке отображаются согласно ветке
